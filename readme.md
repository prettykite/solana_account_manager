# Solana Account Manager

This project implements a TypeScript class to fetch and compare token balances from two Solana accounts.

## Setup

1. Clone the repository.
2. Install dependencies:
  ```bash
  npm install
  ```
3. Run the script:
  ```bash
  npm start
  ```
4. Run the tests:
  ```bash
  npm test
  ```
